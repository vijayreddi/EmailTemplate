Modern World Data Engineering Phase 2 Data River

Paresh Kakadiya

The talk session on "Modern World Data Engineering - River of Data" covered various aspects of data ingestion. Key points included different data ingestion types such as pulling data from sources, pushing data, connecting to databases, receiving event notifications, using SFTP connections, and receiving email notifications with attachments. The different collection of data is used for the data ingestion process which is the first step of data engineering. Different rivers create an ocean, with different types of data making Datalake. That is a raw ingredient for data engineering.
